# Images Folder

Drop all client photos here when ready.

## Naming Convention

Use lowercase, hyphenated filenames that describe the content clearly for SEO:

### Hero Photos (large, 1600×1200px minimum)
- `hero-homepage.jpg`
- `hero-dog-walking.jpg`
- `hero-pet-sitting.jpg`
- `hero-dog-training.jpg`
- `hero-wedding-pet.jpg`
- `hero-snowbird.jpg`

### Team / Founder Photos
- `christa-pecha-founder.jpg`
- `christa-with-dog.jpg`

### Service Photos
- `dog-walking-sarasota.jpg`
- `dog-walking-neighborhood.jpg`

## Technical Specs

- **Format:** JPG (photos), PNG (logos/graphics with transparency)
- **Hero images:** 1600 × 1200px minimum
- **Section photos:** 1200 × 900px
- **File size:** Under 400KB each (use https://squoosh.app to compress)

## Replacing Placeholder Photos

Search each HTML file for `images.unsplash.com` and replace with your image paths.

Example:
```
FIND:    src="https://images.unsplash.com/photo-xxx"
REPLACE: src="assets/images/dog-walking-sarasota.jpg"
```
